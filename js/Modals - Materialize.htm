Introdução

Use um modal para caixas de diálogo, mensagens de confirmações ou outros conteúdos que poderão ser chamados. Para que seu modal funcione você precisa adicionar um id para que o link seja disparado. Para adicionar um botão de fechar, adicione a classe .modal-close no seu botão.

Modals HTML Structure


  <!-- Modal Trigger -->
  <a class="waves-effect waves-light btn modal-trigger" href="#modal1">Modal</a>

  <!-- Modal Structure -->
  <div id="modal1" class="modal">
    <div class="modal-content">
      <h4>Modal Header</h4>
      <p>A bunch of text</p>
    </div>
    <div class="modal-footer">
      <a href="#!" class="modal-action modal-close waves-effect waves-green btn-flat">Agree</a>
    </div>
  </div>

Modal com Rodapé Fixo

Se você tem um conteúdo muito longo e quer que a ação do botão seja visível o tempo todo, você pode adicionar a classe modal-fixed-footer no modal. 


  <!-- Modal Trigger -->
  <a class="waves-effect waves-light btn modal-trigger" href="#modal1">Modal</a>

  <!-- Modal Structure -->
  <div id="modal1" class="modal modal-fixed-footer">
    <div class="modal-content">
      <h4>Modal Header</h4>
      <p>A bunch of text</p>
    </div>
    <div class="modal-footer">
      <a href="#!" class="modal-action modal-close waves-effect waves-green btn-flat ">Agree</a>
    </div>
  </div>

Modais de Folhas no Rodapé

Modais de Folhas no Rodapé são ótimos para mostrar ações para o usuário no rodapé da sua página. Eles funcionam como modais. 


  <!-- Modal Trigger -->
  <a class="waves-effect waves-light btn modal-trigger" href="#modal1">Modal</a>

  <!-- Modal Structure -->
  <div id="modal1" class="modal bottom-sheet">
    <div class="modal-content">
      <h4>Modal Header</h4>
      <p>A bunch of text</p>
    </div>
    <div class="modal-footer">
      <a href="#!" class="modal-action modal-close waves-effect waves-green btn-flat">Agree</a>
    </div>
  </div>

Modais com Butão gatilho

Se você preferir usar um botão para abrir o modal, especifique o id do modal no data-target ao invés de um atributo href. 


  <!-- Modal Trigger -->
  <button data-target="modal1" class="btn modal-trigger">Modal</button>

Inicialização do Plugin jQuery

Para abrir um modal use o gatilho:


  $(document).ready(function(){
    // the "href" attribute of the modal trigger must specify the modal ID that wants to be triggered
    $('.modal').modal();
  });

Você também pode abrir modais programando, o código abaixo mostrar como abrir um modal no document ready:


  $('#modal1').modal('open');

Você também pode fechá-lo programando:


  $('#modal1').modal('close');

Opções

Você pode personalizar o comportamento de cada modal usando essas opções. Por exemplo, você pode chama uma função própria para ser executada enquanto o modal é encerrado. Para isso, basta colocar sua função no código de inicialização como mostrado abaixo.


  $('.modal').modal({
      dismissible: true, // Modal can be dismissed by clicking outside of the modal
      opacity: .5, // Opacity of modal background
      inDuration: 300, // Transition in duration
      outDuration: 200, // Transition out duration
      startingTop: '4%', // Starting top style attribute
      endingTop: '10%', // Ending top style attribute
      ready: function(modal, trigger) { // Callback for Modal open. Modal and trigger parameters available.
        alert("Ready");
        console.log(modal, trigger);
      },
      complete: function() { alert('Closed'); } // Callback for Modal close
    }
  );
